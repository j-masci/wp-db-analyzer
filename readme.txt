== Frequently Asked Questions ==

= Does the plugin provide any tools for optimization? =
	No. It does not optimize your website or database, it just reads it.

= Is the plugin meant for non-developers? =
	In many cases, the plugin will be more useful for developers who understand the structure of the WordPress database.
